Gestión básica de registros (CRUD)

Sistema CRUD en Java Aplicación desarrollada en Java que implementa las operaciones básicas de creación, lectura, actualización y eliminación de registros (CRUD), conectada a una base de datos MySQL. El proyecto está orientado a reforzar conceptos fundamentales como la estructura del código, la conexión a bases de datos, el manejo de datos persistentes y la lógica de negocio. Tecnologías utilizadas: Java, MySQL y JDBC.
